# YieldScope De-Demo Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Remove every dishonest or dead demo surface in YieldScope: wire panels whose data already lands live in Supabase (Buckets A+B), make the briefing-driven panels honest (Bucket D), and excise the four panels that can never be real (Bucket C — owner decisions 2026-07-11: remove call-money-intraday, remove LCR/NSFR, remove ALCO decision log, remove Ways & Means + ADP).

**Architecture:** All work is frontend-only in this repo. Data comes from the shared Supabase project (`metric_history`, `metric_history_monthly`, `auction_calendar`, `auction_results`, `briefings`) via the existing seams: `src/lib/econdelta.ts` (`fetchLatest`/`fetchSeries`, throw-on-error per PR #27), `src/lib/auctions.ts`, and the per-page hooks. No EconDelta changes, no DDL, no new tables.

**Tech Stack:** React 19 + Vite + TypeScript, Vitest + @testing-library/react, GitHub Pages deploy via `.github/workflows/deploy.yml`.

## Global Constraints

- Gate for every task: `npm run test:run && npm run lint && npm run build` — run each bare, redirect to a file, check exit code. NEVER pipe a gate through `tail`/`head`/`grep` (house hook blocks it).
- Never fabricate a number: a missing live value renders `—` + `<DemoBadge />` (or an honest empty-state card), never a fixture digit (AGENT_LEARNINGS 2026-07-09, landmine 18).
- `metric_history` vs `metric_history_monthly` routing is automatic via `MONTHLY_METRICS` in `src/lib/econdelta-metrics.ts` — new ids only need a `METRIC` entry (+ set entry if monthly).
- Date-dependent pure functions take `now: Date` as a parameter (existing pattern: `toUpcomingAuctions`) — no `Date.now()` inside logic under test.
- Run `git push` standalone (house push-guard over-matches chained commands). Branch → PR → per-action merge approval; never push main.
- Immutability, no `any`, interfaces for props (house TS rules).

## Out of scope (explicitly NOT this plan)

- Banking fixture panels with no scraped source: NPL-by-segment, Top-10-banks heatmap, deposits-by-ownership donut, NPL slope chart — stay badged demo (follow-up: needs a BB FSR scrape decision).
- Fiscal-pressure composite index (68/100) — stays badged demo (composite methodology undefined).
- Offline/no-credentials fixture fallbacks for the curve and auctions (`FX.curve`, `FX.auctions`) — by design (F3/F4).
- Dead "Export"/"Filter"/"All 60 banks"/"Export brief"/"Open full briefing" buttons — separate audit item R3.
- `15Y` ladder rung via `yield_15y_monthly` — follow-up; avoids `TenorKey` type ripple through YieldCurve.
- m2 upstream staleness (frozen Feb-2026 at source) — EconDelta grace-window question, not a YieldScope bug.

Change `function tenorRankDays` to `export function tenorRankDays` (needed by the bill/bond split), then append:

```typescript
export interface WeeklyIssuance {
  weekLabel: string  // 'w/c' start date of the 7-day window, e.g. '11 Jul'
  tbillCr: number
  tbondCr: number
}

const DAY_MS = 86_400_000

/** Bucket forward calendar entries into 7-day windows from `now` (today inclusive).
 *  Bills = tenor < 365 days; bonds = the rest. Trailing all-zero weeks are trimmed
 *  so the chart only claims the horizon the BB calendar actually covers. */
export function toWeeklyIssuance(
  cal: AuctionCalendarEntry[],
  now: Date,
  weeks = 12,
): WeeklyIssuance[] {
  const todayISO = isoLocal(now)
  const t0 = Date.UTC(now.getFullYear(), now.getMonth(), now.getDate())
  const buckets: WeeklyIssuance[] = Array.from({ length: weeks }, (_, i) => {
    const start = new Date(t0 + i * 7 * DAY_MS)
    return { weekLabel: dayMon(start.toISOString().slice(0, 10), false), tbillCr: 0, tbondCr: 0 }
  })
  for (const e of cal) {
    if (e.date < todayISO || e.notionalCr == null) continue
    const [y, m, d] = e.date.split('-').map(Number)
    const idx = Math.floor((Date.UTC(y, m - 1, d) - t0) / (7 * DAY_MS))
    if (idx < 0 || idx >= weeks) continue
    if (tenorRankDays(e.tenor) < 365) buckets[idx] = { ...buckets[idx], tbillCr: buckets[idx].tbillCr + e.notionalCr }
    else buckets[idx] = { ...buckets[idx], tbondCr: buckets[idx].tbondCr + e.notionalCr }
  }
  let last = -1
  buckets.forEach((b, i) => { if (b.tbillCr > 0 || b.tbondCr > 0) last = i })
  return buckets.slice(0, last + 1)
}
```

```typescript
// src/hooks/useIssuance.ts
import { useEffect, useState } from 'react'
import { fetchAuctionCalendar, toWeeklyIssuance, type WeeklyIssuance } from '../lib/auctions'

interface UseIssuanceResult {
  data: WeeklyIssuance[] | null
  loading: boolean
  error: Error | null
}

export function useIssuance(): UseIssuanceResult {
  const [state, setState] = useState<UseIssuanceResult>({ data: null, loading: true, error: null })

  useEffect(() => {
    let cancelled = false
    ;(async () => {
      try {
        const cal = await fetchAuctionCalendar(60)
        if (cancelled) return
        setState({ data: toWeeklyIssuance(cal, new Date()), loading: false, error: null })
      } catch (e) {
        if (cancelled) return
        setState({ data: null, loading: false, error: e as Error })
      }
    })()
    return () => { cancelled = true }
  }, [])

  return state
}
```

```typescript
interface TimelineEvent { pct: number; label: string; sub: string; sev: 'info' | 'warn' | 'pos' | 'neg' }

/** Newest-first briefings → oldest-left timeline. Severity = anomaly count
 *  (0 info · 1–3 warn · 4+ neg). Null under 2 points — a one-dot timeline lies. */
export function briefingsToTimeline(briefings: Briefing[]): { axis: string[]; events: TimelineEvent[] } | null {
  const asc = briefings.slice(0, 7).reverse()
  if (asc.length < 2) return null
  const events = asc.map((b, i) => {
    const n = b.featuredAnomalies.length
    return {
      pct: 0.04 + (0.92 * i) / (asc.length - 1),
      label: b.title.length > 44 ? `${b.title.slice(0, 43)}…` : b.title,
      sub: dayMon(b.weekOf, false),
      sev: (n === 0 ? 'info' : n <= 3 ? 'warn' : 'neg') as TimelineEvent['sev'],
    }
  })
  return { axis: asc.map(b => dayMon(b.weekOf, false)), events }
}
```
